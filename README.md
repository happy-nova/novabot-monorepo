# novabot-monorepo

Production-ready Turborepo monorepo scaffold for novabot.sh projects.

## Apps

- `apps/novabot`: Main website (`novabot.sh`) using Next.js 15 + React 19
- `apps/pulsar`: Energy monitoring app (`pulsar.novabot.sh`) using Next.js 15 + React 19 + Tailwind CSS 4

## Packages

- `packages/ui`: Shared React UI components
- `packages/config-eslint`: Shared ESLint configs
- `packages/config-typescript`: Shared TypeScript config presets

## Commands

- `pnpm install`
- `pnpm dev`
- `pnpm build`
- `pnpm lint`
- `pnpm typecheck`
